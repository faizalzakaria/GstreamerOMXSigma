gstreamer Elements

fakesrc - Generate (meaningless) buffers
fakesink - black hole for data
fdsrc - read from a unix file descriptor
filesrc - read from arbitrary point in a file
filesink - write stream to a file
identity - Pass data without modification
queue - Simple asynchronous data queue.
tee - 1-to-N pipe fitting
typefind - Finds the media type of a stream
